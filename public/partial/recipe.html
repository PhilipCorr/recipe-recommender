Chosen Recipe:

We have access to the chosen recipe here. This is obtained using the ng-click class in the anchor tag at the last section of home.html. We will need to call getData and pass the id into the url in order to get more detailed info on the chosenRecipe. This will then be displayed to the user on this page. For info on how to structure the get request see the way that we make a url request when the GET RECIPES button is clicked.
